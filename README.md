# Macro-Tracker
App para el seguimiento de Macros 
------------------------------------
Esto es una app web creada para el seguimiento de macros de forma facil y eficiente

PASOS:

1º Abra el enlace con Safari en el iPhone

2º Pulse en los 3 puntos encontrados en la parte derecha inferior

3º Haga click en el boton de compartir

4º Deslice hacia abajo hasta encontrar la funcion de añadir a la pantalla de inicio

5º Añadala con el nombre que desee y asegurese de que esta marcada la opcion de abrir como app web

6º Salgase de Safari tras añadir la app

7º Abra la aplicacion desde la pantalla de inicio del iPhone 

8º Siga los pasos de la aplicacion para empezar a usarla

9º Añada la clave API donde y cuando se indique. Puedes crear la tuya aqui: https://aistudio.google.com/app/apikey

10º Tras copiarla y pegarla donde se indica en la app ya tendras la app lista para funcionar
